# DeskPilot 视觉设计稿

> 本文档是 DeskPilot 的视觉设计规范，包含浅色/深色主题设计、核心界面设计、交互流程设计。

---

## 1. 设计原则

### 1.1 核心关键词

**简洁 · 优雅 · 高级 · 易用 · 克制 · 信息层级清晰**

### 1.2 设计参考

1. Windows 11 Fluent Design System
2. Raycast 的命令输入体验
3. Notion 的卡片信息结构
4. Linear 的清爽信息层级

### 1.3 避免做

- 复杂动效
- 大面积毛玻璃
- 花哨背景
- 拟物风格
- 强运营感
- 大面积高饱和色
- 复杂 3D 动效
- 卡通化视觉

## 6. 主窗口视觉设计

### 6.1 窗口布局

```
┌─────────────────────────────────────────────────────────┐
│  标题栏（32px，可拖动区域）                    [─][□][×] │
├────────┬────────────────────────────────────────────────┤
│        │                                                │
│  侧边  │              内容区域                          │
│  导航  │                                                │
│  栏    │  (最小宽度 600px，最大宽度无限制)              │
│        │                                                │
│ 180px  │                                                │
│        │                                                │
│        │                                                │
├────────┴────────────────────────────────────────────────┤
│  状态栏（24px）：版本号 · 连接状态              [设置]   │
└─────────────────────────────────────────────────────────┘
```

**浅色主题侧边栏：**
- 背景：`--bg-secondary` (#F5F5F5)
- 当前项背景：`--accent-muted` (#E6F2FB)
- 当前项文字：`--accent-primary` (#0078D4)
- 其他项文字：`--text-secondary` (#666666)
- hover：`--bg-hover` (#E5E5E5)

**深色主题侧边栏：**
- 背景：`--bg-secondary` (#2D2D2D)
- 当前项背景：`--accent-muted` (#0C3B5A)
- 当前项文字：`--accent-primary` (#60CDFF)
- 其他项文字：`--text-secondary` (#B3B3B3)
- hover：`--bg-hover` (#424242)

### 6.2 导航图标

使用 Segoe Fluent Icons 或等效图标：

```
首页      Home                    🏠 (或 Fluent 图标)
桌面整理   Folder                  📁
AI 整理    ChatBot                 💬
整理日志   History                 📋
飞书待办   TaskList                ✓
飞书日程   Calendar                📅
设置       Settings                ⚙
```

图标尺寸：20px，stroke-width: 1.5px

### 6.3 卡片设计

**卡片样式：**
- 背景：`--bg-primary` (#FFFFFF / #1E1E1E)
- 圆角：`--radius-md` (8px)
- 边框：无（纯卡片可加 subtle shadow）
- 阴影：`--shadow-sm`（非必要不添加）
- 内边距：`--space-4` (16px)

**卡片hover（可交互卡片）：**
- 边框：`1px solid --accent-primary`
- 或阴影：`--shadow-md`

## 19. 视觉设计总结

### 19.1 核心原则

1. **Windows 11 Fluent Design** — 遵循系统设计语言
2. **克制** — 不添加不必要的装饰
3. **清晰** — 信息层级明确，不混淆
4. **一致** — 所有界面遵循同一设计语言

### 19.2 浅色 vs 深色核心差异

| 元素 | 浅色 | 深色 |
|------|------|------|
| 主背景 | #FFFFFF | #1E1E1E |
| 次级背景 | #F5F5F5 | #2D2D2D |
| 主文字 | #1A1A1A | #FFFFFF |
| 次级文字 | #666666 | #B3B3B3 |
| 强调色 | #0078D4 | #60CDFF |
| 边框 | #E5E5E5 | #404040 |

### 19.3 设计验收检查

- [ ] 所有文本在浅色/深色主题下可读
- [ ] 功能色（成功/警告/错误）符合语义
- [ ] 交互元素有明确的 hover/active 状态
- [ ] 加载和错误状态有清晰的设计
- [ ] 组件尺寸在触控和鼠标场景下可用
- [ ] 图标和字体在所有支持的 Windows 版本上显示正常
